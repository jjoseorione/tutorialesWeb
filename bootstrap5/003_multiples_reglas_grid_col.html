Primera fila

Sólo tiene 3 modos distintos.

Columna 1

  • 10 columnas si la pantalla mide 1400px o más (límite para xxl
  • 7 columnas si la pantalla mide 768px o más (límite para md)
  • 12 columnas si la pantalla mide menos de 768px (colapsa si es menor a md)

Columna 2

  • 2 columnas si la pantalla mide 1400px o más (límite para xxl
  • 5 columnas si la pantalla mide 768px o más (límite para md)
  • 12 columnas si la pantalla mide menos de 768px (colapsa)

Segunda fila

Tiene 6 modos distintos. La segunda columna trata siempre de mantener su tamaño aunque la primera se haga pequeña. Nunca colapsan.

Columna 1

  • 10 columnas para pantalla >= 1400px
  • 9 columnas para pantalla >= 1200px
  • 8 columnas para pantalla >= 992px
  • 7 columnas para pantalla >= 768px
  • 6 columnas para pantalla >= 576px
  • 5 columnas para pantalla < 576px

Columna 2

  • 2 columnas para pantalla >= 1400px
  • 3 columnas para pantalla >= 1200px
  • 4 columnas para pantalla >= 992px
  • 5 columnas para pantalla >= 768px
  • 6 columnas para pantalla >= 576px
  • 7 columnas para pantalla < 576px
