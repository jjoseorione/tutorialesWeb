1. ¿Qué es JQuery?

JQuery es una biblioteca de JavaScript para acceder a los objetos del DOM de un modo simplificado. Para descargarla o ver las mejoras, podemos entrar a www.jquery.com. Las páginas web modernas como realizar drag and drop, auto-completar, animaciones, etc y desarrollar todos estos conceptos desde cero puede resultar complicado, sobretodo si tenemos que presentar la solución con muy poco tiempo. En este tipo de situaciones, el empleo de bibliotecas como JQuery facilitan el desarrollo de la aplicación.

Para utilizar la biblioteca, debemos descargarla del sitio oficial y en cada página que lo requiera, agregar:

<script src="jquery-3.4.1.min.js"></script>

También es posible realizar un CDN con la liga http de donde se almacene la biblioteca:

<script src="https://code.jquery.com/jquery-3.4.1.min.js"></script>
